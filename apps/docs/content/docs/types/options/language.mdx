---
title: Language
description: Type definitions for language and translation codes used across the TMDB client.
---

These types represent the language system used throughout the TMDB API.
TMDB primarily uses [ISO 639-1](https://en.wikipedia.org/wiki/ISO_639-1) codes, optionally paired
with an [ISO 3166-1](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) country code to form locale strings.

> **TMDB Reference:** [Languages overview](https://developer.themoviedb.org/docs/languages) ·
> [Configuration: Languages endpoint](https://developer.themoviedb.org/reference/configuration-languages)

---

## `Language`

The main language type accepted across all client methods and constructor options.
It is a union of [`PrimaryTranslations`](#primarytranslations) and [`LanguageISO6391`](#languageiso6391).

```ts
type Language = PrimaryTranslations | LanguageISO6391;
```

Prefer [`PrimaryTranslations`](#primarytranslations) locale codes (e.g., `"it-IT"`, `"en-US"`) wherever possible,
as they map to TMDB's fully-supported translation set and return richer localized data.
Use bare [`LanguageISO6391`](#languageiso6391) codes only when a primary translation is not available
for your target language.

## `PrimaryTranslations`

A union of locale strings in `language-REGION` format that TMDB has dedicated, curated translation
support for. These are the recommended values to use for the `language` option.

> The full union contains ~80 locale codes. See the
> [TMDB primary translations reference](https://developer.themoviedb.org/reference/configuration-primary-translations)
> for the authoritative list, or inspect the `PrimaryTranslations` type directly in the package source.

### Format

Locale strings follow the `language-REGION` format, combining an
[ISO 639-1](https://en.wikipedia.org/wiki/ISO_639-1) language code with an
[ISO 3166-1 alpha-2](https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2) country code:

```
it-IT   →  Italian, Italy
en-US   →  English, United States
pt-BR   →  Portuguese, Brazil
```

## `LanguageISO6391`

A union of bare [ISO 639-1](https://en.wikipedia.org/wiki/ISO_639-1) two-letter language codes.
These are supported by TMDB but may return incomplete or untranslated data for some fields,
as they don't map to a specific regional translation set.

```ts
type LanguageISO6391 =
	| "en" // English
	| "it" // Italian
	| "fr" // French
	| "de" // German
	| "es" // Spanish
	| "pt" // Portuguese
	| "ja" // Japanese
	| "ko" // Korean
	| "zh" // Chinese
	| "ru" // Russian
	| "ar" // Arabic
	| "hi" // Hindi
	| "tr" // Turkish
	| "nl" // Dutch
	| "pl" // Polish
	| "sv" // Swedish
	| "uk"; // Ukrainian
// ... and more
```

> The full union contains 180+ ISO 639-1 codes. Note that some codes here (e.g., `"xx"`)
> are non-standard placeholders used by TMDB for content with no defined language.
> See the [TMDB Languages endpoint](https://developer.themoviedb.org/reference/configuration-languages)
> for the complete list. We also provide a method for fetching the configuration endpoint directly in our
> package.
