## `GuestSessionRatedMoviesResponse`

The response type from [`guest_sessions.rated_movies()`](/docs/api-reference/guest-sessions/rated-movies).

A [`PaginatedResponse<AccountRatedMovieItem>`](/docs/types/account#accountratedmovieitem) — each result includes a `rating` field with the user-assigned score alongside the standard movie fields.

## `GuestSessionRatedTVResponse`

The response type from [`guest_sessions.rated_tv()`](/docs/api-reference/guest-sessions/rated-tv).

A [`PaginatedResponse<AccountRatedTVItem>`](/docs/types/account#accountratedtvitem) — each result includes a `rating` field alongside the standard TV show fields.

## `GuestSessionRatedEpisodesResponse`

The response type from [`guest_sessions.rated_tv_episodes()`](/docs/api-reference/guest-sessions/rated-tv-episodes).

A [`PaginatedResponse<AccountRatedEpisodeItem>`](/docs/types/account#accountratedepisodeitem) — each result includes a `rating` field alongside episode-level fields such as `air_date`, `episode_number`, `season_number`, and `show_id`.
