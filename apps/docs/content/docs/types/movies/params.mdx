## `MovieListParams`

Parameters accepted by all movie list endpoints: [`now_playing()`](/docs/api-reference/movie-lists/now-playing),
[`popular()`](/docs/api-reference/movie-lists/popular), [`top_rated()`](/docs/api-reference/movie-lists/top-rated),
and [`upcoming()`](/docs/api-reference/movie-lists/upcoming).

This is an alias of [`TMDBQueryParams`](/docs/types/common/params#tmdbqueryparams).
