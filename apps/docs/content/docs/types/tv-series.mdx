These types represent the full TV series domain — details, appended sub-resources, and query parameters.

## `TVSeriesDetails`

The complete representation of a TV show as returned by [`tv_series.details()`](/docs/api-reference/tv-series/details).

<AutoTypeTable path="tv-series.ts" name="TVSeriesDetails" />

---

## `TVDetailsWithAppends<T>`

Extends [`TVSeriesDetails`](#tvseriesdetails) with one or more appended sub-resources.
`T` is a readonly array of [`TVAppendToResponseNamespace`](#tvappendtoresponsenamespace) keys.

```ts
const show: TVDetailsWithAppends<["credits", "keywords"]> = await tmdb.tv_series.details({
	series_id: 1396,
	append_to_response: ["credits", "keywords"],
});
```

---

## `TVAppendToResponseNamespace`

Union of all valid keys that can be passed to `append_to_response`.

```ts
type TVAppendToResponseNamespace =
	| "credits"
	| "external_ids"
	| "images"
	| "keywords"
	| "recommendations"
	| "similar"
	| "translations"
	| "videos";
```
