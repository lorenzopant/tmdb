## `TVSeasonDetailsWithAppends<T>`

Extends [`TVSeason`](#tvseason) with one or more appended sub-resources.
`T` is a readonly array of [`TVSeasonAppendToResponseNamespace`](#tvseasonappendtoresponsenamespace) keys.

```ts
const season: TVSeasonDetailsWithAppends<["credits", "images"]> = await tmdb.tv_seasons.details({
	series_id: 1396,
	season_number: 1,
	append_to_response: ["credits", "images"],
});
```

---

## `TVSeasonAppendToResponseNamespace`

Union of all valid keys that can be passed to `append_to_response`.

```ts
type TVSeasonAppendToResponseNamespace =
	| "aggregate_credits"
	| "credits"
	| "external_ids"
	| "images"
	| "translations"
	| "videos"
	| "watch_providers";
```
