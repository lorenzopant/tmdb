## `TMDBLoggerFn`

A callback function that receives a [`TMDBLoggerEntry`](#tmdbloggerentry) for every request,
response, and network error fired by the TMDB client.

```ts
type TMDBLoggerFn = (entry: TMDBLoggerEntry) => void;
```

Pass it to the `logger` option when creating a `TMDB` instance:

```ts
import { TMDB, TMDBLoggerFn } from "@lorenzopant/tmdb";

const myLogger: TMDBLoggerFn = (entry) => {
	console.log(entry.type, entry.endpoint);
};

const tmdb = new TMDB("your-api-key", { logger: myLogger });
```

---

## `TMDBLoggerEntry`

The object passed to your [`TMDBLoggerFn`](#tmdblognerfn) on every log event.

```ts
type TMDBLoggerEntry = {
	type: "request" | "response" | "error";
	method: "GET";
	endpoint: string;
	status?: number;
	statusText?: string;
	durationMs?: number;
	tmdbStatusCode?: number;
	errorMessage?: string;
};
```
