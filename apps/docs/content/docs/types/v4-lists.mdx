## `V4ListMediaType`

Union type for list item media types.

```ts
type V4ListMediaType = "movie" | "tv";
```

## `V4ListResult`

A single media item returned inside the `results` array of [`V4ListDetails`](#v4listdetails).
Fields vary between movies and TV shows — movie-specific fields (`title`, `release_date`, `video`)
are optional and present only for `media_type: "movie"`; TV-specific fields (`name`,
`first_air_date`, `origin_country`) are present only for `media_type: "tv"`.
