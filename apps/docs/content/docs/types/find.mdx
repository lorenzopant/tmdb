<TypeTable
	type={{

	}}
/>

---

### `FindExternalSource`

Supported external source values accepted by [`find.by_id()`](/docs/api-reference/find/by-id).

```ts
type FindExternalSource =
	| "imdb_id"
	| "tvdb_id"
	| "freebase_mid"
	| "freebase_id"
	| "tvrage_id"
	| "facebook_id"
	| "instagram_id"
	| "twitter_id";
```
