Get the paginated list of TV shows rated during a guest session.

```ts
async rated_tv(params: GuestSessionRatedParams): Promise<GuestSessionRatedTVResponse>
```

> **TMDB Reference:** [Rated TV](https://developer.themoviedb.org/reference/guest-session-rated-tv)

## Parameters

| Name               | Type                                                | Required | Description                                                                                     |
| ------------------ | --------------------------------------------------- | -------- | ----------------------------------------------------------------------------------------------- |
| `guest_session_id` | `string`                                            | ✅       | The guest session ID returned by `authentication.create_guest_session()`.                       |
| `language`         | [`Language`](/docs/types/options/language#language) | ❌       | Language for localized results. Defaults to `en-US`.                                            |
| `page`             | `number`                                            | ❌       | Page number to return. Defaults to `1`.                                                         |
| `sort_by`          | `AccountSortBy`                                     | ❌       | Sort direction. One of `"created_at.asc"` or `"created_at.desc"`. Defaults to `created_at.asc`. |

## Returns

A [`GuestSessionRatedTVResponse`](/docs/types/guest-sessions#guestsessionratedtvresponse) — a [`PaginatedResponse<AccountRatedTVItem>`](/docs/types/account#accountratedtvitem) with `page`, `total_pages`, `total_results`, and an array of rated TV show objects, each including a `rating` field set by the guest user.

## Example

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-api-key");
const shows = await tmdb.guest_sessions.rated_tv({
	guest_session_id: "abc123",
	language: "en-US",
});

console.log(shows.results[0].name);
// "Game of Thrones"
console.log(shows.results[0].rating);
// 8.5
```
