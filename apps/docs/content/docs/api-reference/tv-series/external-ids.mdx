Get a list of external IDs for a TV show.
Supported platforms include IMDb, TVDB, Wikidata, Freebase (deprecated), TVRage (discontinued), Facebook, Instagram, and Twitter.

```ts
async external_ids(params: TVBaseParam): Promise<TVExternalIDs>
```

> **TMDB Reference:** [TV Series External IDs](https://developer.themoviedb.org/reference/tv-series-external-ids)

## Parameters

| Name        | Type     | Required | Description                |
| ----------- | -------- | -------- | -------------------------- |
| `series_id` | `number` | âœ…       | TMDB TV series identifier. |

## Returns

[`TVExternalIDs`](/docs/types/tv-series#tvexternalids)

## Example

```ts
const ids = await tmdb.tvSeries.external_ids({ series_id: 1396 });
console.log(ids.imdb_id); // "tt0903747"
console.log(ids.tvdb_id); // 81189
```
