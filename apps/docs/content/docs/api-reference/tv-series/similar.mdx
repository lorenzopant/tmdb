Get a list of TV shows similar to the given series.

```ts
async similar(params: TVSimilarParams): Promise<TVSimilar>
```

> **TMDB Reference:** [TV Series Similar](https://developer.themoviedb.org/reference/tv-series-similar)

## Parameters

| Name        | Type                                                | Required | Description                                          |
| ----------- | --------------------------------------------------- | -------- | ---------------------------------------------------- |
| `series_id` | `number`                                            | ✅       | TMDB TV series identifier.                           |
| `language`  | [`Language`](/docs/types/options/language#language) | ❌       | Language for localized results. Defaults to `en-US`. |
| `page`      | `number`                                            | ❌       | Page number for paginated results. Defaults to `1`.  |

## Returns

[`TVSimilar`](/docs/types/tv-series#tvsimilar) —
a [`PaginatedResponse<TVSeriesResultItem>`](/docs/types/search#tvseriesresultitem).

## Example

```ts
const { results } = await tmdb.tvSeries.similar({ series_id: 1396 });
console.log(results.map((s) => s.name));
```
