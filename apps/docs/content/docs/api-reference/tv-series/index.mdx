---
title: TV Series
description: Fetch TV series details, credits, images, videos, ratings, and more from the TMDB catalog.
---

The `TVSeriesAPI` provides access to all TV series-related data from TMDB —
from full details and credits to images, videos, and streaming availability.

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-api-key");
const show = await tmdb.tvSeries.details({ series_id: 1396 });
```

## Methods

- [`details()`](/docs/api-reference/tv-series/details) — Get the top level details of a TV series by ID. Supports `append_to_response`.
- [`aggregate_credits()`](/docs/api-reference/tv-series/aggregate-credits) — Get the full cast and crew across all seasons and episodes.
- [`alternative_titles()`](/docs/api-reference/tv-series/alternative-titles) — Get alternative titles for a TV show across different countries and languages.
- [`changes()`](/docs/api-reference/tv-series/changes) — Get the change history for a TV show (up to 14 days).
- [`content_ratings()`](/docs/api-reference/tv-series/content-ratings) — Get content ratings per country for a TV show.
- [`credits()`](/docs/api-reference/tv-series/credits) — Get the latest season cast and crew for a TV show.
- [`episode_groups()`](/docs/api-reference/tv-series/episode-groups) — Get the episode groups that have been added to a TV show.
- [`external_ids()`](/docs/api-reference/tv-series/external-ids) — Get external platform identifiers (IMDb, TVDB, Wikidata, social media).
- [`images()`](/docs/api-reference/tv-series/images) — Get backdrops, logos, and posters for a TV show.
- [`keywords()`](/docs/api-reference/tv-series/keywords) — Get the keywords associated with a TV show.
- [`latest()`](/docs/api-reference/tv-series/latest) — Get the most recently added TV show on TMDB.
- [`lists()`](/docs/api-reference/tv-series/lists) — Get the lists that a TV series has been added to.
- [`recommendations()`](/docs/api-reference/tv-series/recommendations) — Get recommended TV shows based on a series.
- [`reviews()`](/docs/api-reference/tv-series/reviews) — Get user reviews for a TV show.
- [`screened_theatrically()`](/docs/api-reference/tv-series/screened-theatrically) — Get episodes that have been screened theatrically.
- [`similar()`](/docs/api-reference/tv-series/similar) — Get TV shows similar to a given series.
- [`translations()`](/docs/api-reference/tv-series/translations) — Get all available translations for a TV show.
- [`videos()`](/docs/api-reference/tv-series/videos) — Get trailers, teasers, and clips for a TV show.
- [`watch_providers()`](/docs/api-reference/tv-series/watch-providers) — Get streaming, rental, and purchase availability per country.
