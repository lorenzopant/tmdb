---
title: Upcoming
description: Get a list of movies that are being released soon.
---

Returns a paginated list of movies that are scheduled for upcoming theatrical release.
Results are filtered by region when a `region` parameter is provided.

```ts
async upcoming(params?: MovieListParams): Promise<PaginatedResponse<MovieResultItem>>
```

> **TMDB Reference:** [Movie Upcoming](https://developer.themoviedb.org/reference/movie-upcoming-list)

## Parameters

| Name       | Type                                                             | Required | Description                                                       |
| ---------- | ---------------------------------------------------------------- | -------- | ----------------------------------------------------------------- |
| `language` | [`Language`](/docs/types/options/language#language)              | ❌       | Language for localized titles and overviews. Defaults to `en-US`. |
| `page`     | `number`                                                         | ❌       | Page number for paginated results. Defaults to `1`.               |
| `region`   | [`CountryISO3166_1`](/docs/types/options/region#countryiso31661) | ❌       | ISO 3166-1 country code to filter results by country.             |

## Returns

A [`PaginatedResponse<MovieResultItem>`](/docs/types/common/pagination#paginatedresponse) containing
an array of [`MovieResultItem`](/docs/types/movies#movieresultitem) objects.

## Example

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-api-key");
const { results } = await tmdb.movieLists.upcoming();

// With params
const italian = await tmdb.movieLists.upcoming({ language: "it-IT", region: "IT" });
```
