Add a movie to a list. Requires a valid v3 session ID.

```ts
async add_movie(params: ListMutationParams, body: ListMovieBody): Promise<ListMutationResponse>
```

> **TMDB Reference:** [Add Movie](https://developer.themoviedb.org/reference/list-add-movie)

## Parameters

| Name         | Type     | Required | Description    |
| ------------ | -------- | -------- | -------------- |
| `list_id`    | `number` | ✅       | TMDB list ID.  |
| `session_id` | `string` | ✅       | v3 session ID. |

## Body

| Name       | Type     | Required | Description                       |
| ---------- | -------- | -------- | --------------------------------- |
| `media_id` | `number` | ✅       | TMDB movie ID to add to the list. |

## Returns

[`ListMutationResponse`](/docs/types/lists#listmutationresponse)

## Example

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-api-key");
const result = await tmdb.lists.add_movie(
	{ list_id: 5861, session_id: "your-session-id" },
	{ media_id: 550 },
);

console.log(result.status_code); // 12
```
