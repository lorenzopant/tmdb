Remove a movie from a list. Requires a valid v3 session ID.

```ts
async remove_movie(params: ListMutationParams, body: ListMovieBody): Promise<ListMutationResponse>
```

> **TMDB Reference:** [Remove Movie](https://developer.themoviedb.org/reference/list-remove-movie)

## Parameters

| Name         | Type     | Required | Description    |
| ------------ | -------- | -------- | -------------- |
| `list_id`    | `number` | ✅       | TMDB list ID.  |
| `session_id` | `string` | ✅       | v3 session ID. |

## Body

| Name       | Type     | Required | Description                            |
| ---------- | -------- | -------- | -------------------------------------- |
| `media_id` | `number` | ✅       | TMDB movie ID to remove from the list. |

## Returns

[`ListMutationResponse`](/docs/types/lists#listmutationresponse)

## Example

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-api-key");
const result = await tmdb.lists.remove_movie(
	{ list_id: 5861, session_id: "your-session-id" },
	{ media_id: 550 },
);

console.log(result.status_code); // 13
```
