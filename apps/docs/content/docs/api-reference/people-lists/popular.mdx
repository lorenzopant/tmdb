Returns a paginated list of people ordered by TMDB's popularity score.

```ts
async popular(params?: PeopleListParams): Promise<PaginatedResponse<PersonResultItem>>
```

> **TMDB Reference:** [Person Popular List](https://developer.themoviedb.org/reference/person-popular-list)

## Parameters

| Name       | Type                                                | Required | Description                                          |
| ---------- | --------------------------------------------------- | -------- | ---------------------------------------------------- |
| `language` | [`Language`](/docs/types/options/language#language) | ❌       | Language for localized results. Defaults to `en-US`. |
| `page`     | `number`                                            | ❌       | Page number for paginated results. Defaults to `1`.  |

## Returns

A [`PaginatedResponse<PersonResultItem>`](/docs/types/common#paginatedresponset) containing
an array of [`PersonResultItem`](/docs/types/search#personresultitem) objects.

## Example

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-api-key");
const { results } = await tmdb.people_lists.popular();

// With params
const page2 = await tmdb.people_lists.popular({ page: 2, language: "fr-FR" });
```
