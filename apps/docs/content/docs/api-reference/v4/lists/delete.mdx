Permanently deletes a list and all of its items. **This action cannot be undone.**
If you only want to empty the list, use [`clear()`](/docs/api-reference/v4/lists/clear) instead.

```ts
async delete(params: V4DeleteListParams): Promise<V4ListStatusResponse>
```

> **TMDB Reference:** [Delete List](https://developer.themoviedb.org/reference/list-delete)

## Parameters

<TypeTable
	type={{
		list_id: {
			type: "number",
			required: true,
			description: "The TMDB list ID to delete.",
		},
	}}
/>

## Returns

[`V4ListStatusResponse`](/docs/types/v4-lists#v4liststatusresponse)

## Errors

| HTTP | TMDB code | Cause                               |
| ---- | --------- | ----------------------------------- |
| 401  | 3         | Invalid or missing access token.    |
| 404  | 34        | List not found.                     |
| 403  | 15        | Not authorized to delete this list. |

## Example

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-access-token");

const { success } = await tmdb.v4.lists.delete({ list_id: 8700 });
console.log(success); // true
```
