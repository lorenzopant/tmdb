## Parameters

| Name       | Type                                                | Required | Description                               |
| ---------- | --------------------------------------------------- | -------- | ----------------------------------------- |
| `language` | [`Language`](/docs/types/options/language#language) | ❌       | Language used for localized region names. |

## Returns

[`WatchProviderRegionsResponse`](/docs/types/watch-providers#watchproviderregionsresponse)

## Example

```ts
const regions = await tmdb.watch_providers.available_regions();

console.log(regions.results[0]?.english_name);
```

```json
{
	"results": [

			"iso_3166_1": "US",

			"native_name": "United States"
		}
	]
}
```
