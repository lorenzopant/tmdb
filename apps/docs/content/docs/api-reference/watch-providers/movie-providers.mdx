## Parameters

| Name       | Type                                                | Required | Description                                 |
| ---------- | --------------------------------------------------- | -------- | ------------------------------------------- |
| `language` | [`Language`](/docs/types/options/language#language) | ❌       | Language used for localized provider names. |

## Returns

[`WatchProviderListResponse`](/docs/types/others#watchproviderlistresponse)

## Example

```ts
const providers = await tmdb.watch_providers.movie_providers();

console.log(providers.results[0]?.provider_name);
```
