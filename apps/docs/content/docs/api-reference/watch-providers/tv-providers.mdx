## Parameters

| Name       | Type                                                | Required | Description                                 |
| ---------- | --------------------------------------------------- | -------- | ------------------------------------------- |
| `language` | [`Language`](/docs/types/options/language#language) | ❌       | Language used for localized provider names. |

## Returns

[`WatchProviderListResponse`](/docs/types/watch-providers#watchproviderlistresponse)

## Example

```ts
const providers = await tmdb.watch_providers.tv_providers();

console.log(providers.results[0]?.provider_name);
```
