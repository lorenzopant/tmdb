Get a list of TV shows airing today. The result is anchored to a timezone — use the `timezone` parameter to adjust the definition of "today" to a specific region.

```ts
async airing_today(params?: TVSeriesListParams): Promise<PaginatedResponse<TVSeriesResultItem>>
```

> **TMDB Reference:** [TV Airing Today](https://developer.themoviedb.org/reference/tv-series-airing-today-list)

## Parameters

| Name       | Type                                                | Required | Description                                              |
| ---------- | --------------------------------------------------- | -------- | -------------------------------------------------------- |
| `language` | [`Language`](/docs/types/options/language#language) | ❌       | Language for localized results. Defaults to `en-US`.     |
| `page`     | `number`                                            | ❌       | Page number for paginated results. Defaults to `1`.      |
| `timezone` | `string`                                            | ❌       | Timezone to anchor "today" (e.g., `"America/New_York"`). |

## Returns

[`PaginatedResponse<TVSeriesResultItem>`](/docs/types/search#tvseriesresultitem)

## Example

```ts
const { results } = await tmdb.tvSeriesLists.airing_today({
	language: "en-US",
	timezone: "America/New_York",
});

console.log(results.map((s) => s.name));
```
