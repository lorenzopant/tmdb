---
title: Details
description: Query the details of a specific TV network.
---

```ts
async details(params: NetworkBaseParams): Promise<Network>
```

> **TMDB Reference:** [Network Details](https://developer.themoviedb.org/reference/network-details)

## Parameters

| Name         | Type     | Required | Description              |
| ------------ | -------- | -------- | ------------------------ |
| `network_id` | `number` | ✅       | TMDB network identifier. |

## Returns

A [`Network`](/docs/types/networks#network) object containing
the details of the specified network.

## Example

```ts
import { TMDB } from "@lorenzopant/tmdb";

const tmdb = new TMDB("your-api-key");
const network = await tmdb.networks.details({ network_id: 49 }); // HBO
```
