Get a list of recommended movies for a specific movie, based on popularity and user ratings.

```ts
async recommendations(params: MovieRecommendationsParams): Promise<MovieRecommendations>
```

> **TMDB Reference:** [Movie Recommendations](https://developer.themoviedb.org/reference/movie-recommendations)

## Parameters

| Name       | Type                                       | Required | Description                                          |
| ---------- | ------------------------------------------ | -------- | ---------------------------------------------------- |
| `movie_id` | `number`                                   | ✅       | TMDB movie identifier.                               |
| `language` | [`Language`](/docs/types/options#language) | ❌       | Language for localized results. Defaults to `en-US`. |
| `page`     | `number`                                   | ❌       | Page number for paginated results. Defaults to `1`.  |

## Returns

[`MovieRecommendations`](/docs/types/movies#movierecommendations) —
a [`PaginatedResponse<MovieResultItem>`](/docs/types/common#paginatedresponset).

## Example

```ts
const { results } = await tmdb.movies.recommendations({ movie_id: 550 });
console.log(results.map((m) => m.title));
```
